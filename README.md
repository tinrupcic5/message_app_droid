# MessageApp_droid
Vsite - android 


## Overview
MessageApp_droid is an Android application that allows users to register, login, and chat with other users using the MessageApp REST API. Users can register on the first use of the app, and subsequent logins will automatically log them in if they are registered users. The app supports one-on-one chats with other users and provides a contacts view to manage contacts.

## Features
User Registration: New users can register on their first use of the app by providing necessary details.

Automatic Login: Once registered, users remain logged in every time they open the app without the need for manual login.

Chatting: Users can engage in one-on-one chats with other users who are registered on the platform.

Contacts View: The app provides a contacts view where users can see their contacts and start a chat with them.

## Setup
To set up and run the MessageApp_droid project:

Clone the repository from the GitHub repository MessageApp_droid.
Open the project in Android Studio.
Build and run the project on an Android device or emulator.
